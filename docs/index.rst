Simple-ZCU208-Example Documentation
=====================================

Reference firmware and software example for the Xilinx ZCU208 evaluation
board (Xilinx Zynq UltraScale+ RFSoC, FPGA part
``XCZU48DR-FSVG1517-2-E``). Built on the SLAC ``axi-soc-ultra-plus-core``
platform; shared workflow documentation lives on the
:hub:`platform docs site <index.html>`.
